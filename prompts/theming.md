# Theme assignment rules (Groq call 1)

Assign each review to **exactly one** theme:

1. **Trading & orders** — order execution, charts, F&O, stocks, mutual funds, IPO
2. **KYC & onboarding** — verification, PAN, bank linking, account opening
3. **Deposits & withdrawals** — UPI, bank transfer, settlement delays, balance
4. **App UX & bugs** — crashes, slow app, UI glitches, login failures
5. **Customer support** — refunds, complaints, chatbot, unresponsive care

Return JSON: `{"assignments": [{"id": 0, "theme": "..."}, ...]}`
